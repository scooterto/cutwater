Usage Guidelines

Logo Versions (Only Two)

• Light Background: Navy C (#1a2b5f) - Used in header (light mode only)
• Dark Background: White C (#ffffff) - Used in footer (both modes) and header (dark mode)
• The balance line and dot are always Highlight Blue (#5BA4E6) across all contexts

Typography Hierarchy

• Use Playfair Display for all headings and large numbers
• Use Source Serif 4 for body text, labels, and buttons
• Use Noto Sans exclusively for currency symbols ($) to ensure single-stroke design
• Enable 'lnum' and 'pnum' font features for tabular numbers

Dark Mode Implementation

• Primary text (headings) changes from #1a2b5f to #ffffff
• Logo "C" changes from #1a2b5f to #ffffff
• Background changes from #ffffff to #0F1035
• Cards/elevated surfaces use #1a1b3d
• Borders change from #e8eaed to #365486
• Primary buttons become #5BA4E6 with dark text (#0f1a3d)

Color Application

• Success Green (#2d8a5f) for APY percentages and positive metrics
• Highlight Blue (#1e88e5 light / #5BA4E6 dark) for accents and CTAs
• Body Gray (#5a6577 light / #8B9DC3 dark) for secondary text
• Border Gray (#e8eaed light / #365486 dark) for subtle dividers
